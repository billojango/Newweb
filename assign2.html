PROFOUND GRID

A responsive grid system for fixed and fluid layouts. Built

in SCSS, it gives you flexibility and full control

Responsive

Make the grid fluid or fixed and add per-media layouts as needed. Change the column and gutter width and remove or add columns as needed

Semantic

The grid keeps you markup semantic and works without any "grid_x" classes. Unless you want to use them, in which case they are included

Precise

Profound Grid uses negative margins to calculate columns. Unlike with other grid systems, fluid layouts will look exactly the same in every browser
